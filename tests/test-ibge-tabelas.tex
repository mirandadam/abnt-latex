\chapter{Exemplos auxiliares}

\section{Apresentação de tempo}

% Checklist 12
A série de dados abrange o período de \ibgeSerieConsec{1981}{1985}
(série consecutiva). Para anos não consecutivos, usa-se
\ibgeSerieNConsec{1981}{1985}. Para safras agrícolas, a notação é
\ibgeSafra{1991}{1992}.

\section{Classes de frequência}

% Checklist 11
As classes de frequência podem ser expressas de três formas:
\begin{itemize}
  \item Inclui inferior, exclui superior: \ibgeClasseIE{10}{20}
  \item Exclui inferior, inclui superior: \ibgeClasseEI{10}{20}
  \item Inclui ambos: \ibgeClasseII{10}{20}
\end{itemize}
